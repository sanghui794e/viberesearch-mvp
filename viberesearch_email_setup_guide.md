# 📧 VibeResearch 실제 이메일 자동 발송 연동 1:1 세부 가이드

이 가이드는 대표님이 가장 널리 사용하시는 **Gmail**을 발송용 이메일 서버(SMTP)로 사용하여, VibeResearch에서 가입 승인 시 실제 크리에이터(지인)들의 메일함으로 안내 메일이 실시간 발송되도록 셋업하는 방법을 가장 쉽게 알려주는 매뉴얼입니다.

---

## 1단계. Google 계정에서 16자리 '앱 비밀번호' 발급받기 🔐
Google의 강화된 보안 정책으로 인해, 발송용 계정의 일반 비밀번호를 그대로 쓰면 메일 발송이 거부됩니다. 반드시 아래 순서대로 **전용 앱 비밀번호**를 발급받으셔야 합니다.

1.  **Google 계정 설정 진입**:
    *   [Google 내 계정](https://myaccount.google.com) 페이지로 이동합니다.
2.  **2단계 인증 활성화 (필수)**:
    *   좌측 메뉴에서 **[보안]** 탭을 클릭합니다.
    *   'Google에 로그인하는 방법' 섹션에서 **[2단계 인증]**이 사용으로 되어 있는지 확인합니다. (미설정 상태라면 안내에 따라 2단계 인증을 먼저 활성화해 주세요.)
3.  **앱 비밀번호 검색 및 생성**:
    *   **[보안]** 탭 상단의 **'설정 검색'**창에 `"앱 비밀번호"` 또는 `"App Passwords"`를 검색하여 클릭합니다.
    *   앱 이름 입력란에 알아보기 쉽게 `"VibeResearch Email"` 또는 임의의 이름을 적고 **[만들기]**를 누릅니다.
4.  **16자리 노란색 비밀번호 복사**:
    *   화면에 표시되는 **16자리 영문 키(예: `abcd efgh ijkl mnop`)**를 따로 복사해 메모장에 안전하게 저장해 둡니다. (창을 닫으면 다시 볼 수 없습니다!)

---

## 2단계. 로컬 개발 환경 테스트를 위한 설정 💻
내 컴퓨터에서 실제 메일이 나가는지 먼저 테스트해 보고 싶으실 때 설정하는 방법입니다.

1.  **`.env.local` 파일 생성 및 열기**:
    *   `1인 기업 대표` 폴더 루트에 파일 이름이 정확히 `.env.local`인 파일을 새로 생성합니다. (만약 이미 있다면 기존 파일을 엽니다.)
2.  **아래 내용을 복사해서 붙여넣고 내 정보로 수정하기**:
    *   1단계에서 발급받은 16자리 앱 비밀번호(띄어쓰기 없이 붙여서 작성)와 본인의 Gmail 계정을 아래 양식에 맞게 입력해 줍니다.

```env
# 이메일 자동 발송용 실전 SMTP 연동 설정
EMAIL_SMTP_HOST=smtp.gmail.com
EMAIL_SMTP_PORT=587
EMAIL_SMTP_USER=대표님의_구글_이메일@gmail.com
EMAIL_SMTP_PASS=발급받은_16자리_앱비밀번호
EMAIL_FROM=대표님의_구글_이메일@gmail.com
```

3.  **테스트 실행**:
    *   로컬 개발 서버(`npm run dev`)를 켜둔 상태로, [로컬 어드민 페이지](http://localhost:3000/admin)에 접속하여 회원을 `승인하기` 해봅니다.
    *   수령인의 실제 메일함에 알림 편지가 즉시 도착하는 것을 확인합니다.

---

## 3단계. Vercel(실제 배포 사이트)에 환경 변수 등록하기 🚀
대표님이 지인들에게 공유하신 배포 링크([https://viberesearch-mvp.vercel.app](https://viberesearch-mvp.vercel.app))에서도 실제 메일이 나가도록 만드는 최종 설정입니다.

1.  **Vercel 대시보드 접속**:
    *   [Vercel Dashboard](https://vercel.com/dashboard)에 로그인하신 후, **`viberesearch-mvp`** 프로젝트를 선택하여 클릭합니다.
2.  **프로젝트 설정(Settings) 메뉴 진입**:
    *   상단 탭 중 가장 우측에 있는 **[Settings]** 탭을 클릭합니다.
3.  **환경 변수(Environment Variables) 메뉴 이동**:
    *   좌측 사이드바에서 **[Environment Variables]** 항목을 클릭합니다.
4.  **변수값 차례대로 등록하기 (총 5개)**:
    *   오른쪽 화면의 **Key**와 **Value** 란에 아래 표의 내용들을 하나씩 채우고 우측의 **[Add]** 버튼을 각각 눌러 추가해 줍니다.

| 변수 이름 (Key) | 등록할 값 (Value) | 설명 |
| :--- | :--- | :--- |
| **`EMAIL_SMTP_HOST`** | `smtp.gmail.com` | Google 메일 서버 주소 |
| **`EMAIL_SMTP_PORT`** | `587` | 통신용 포트 번호 |
| **`EMAIL_SMTP_USER`** | `대표님의_구글_이메일@gmail.com` | 발송에 사용할 Gmail 계정 |
| **`EMAIL_SMTP_PASS`** | `발급받은_16자리_앱비밀번호` | 1단계에서 얻은 16자리 키 |
| **`EMAIL_FROM`** | `대표님의_구글_이메일@gmail.com` | 수신자 메일함에 보낸이로 표시될 주소 |

5.  **배포 사이트에 환경 변수 최종 적용하기 (Redeploy)**:
    *   Vercel은 환경 변수가 추가되면 웹사이트를 한 번 새로 빌드(재배포)해 주어야 값이 온전히 반영됩니다.
    *   상단 탭 중 **[Deployments]**를 클릭합니다.
    *   가장 최근 빌드 내역 우측에 있는 **점 3개 단추(...)**를 누르고 **[Redeploy]**를 클릭하여 재빌드를 진행해 주시면 적용이 끝납니다!
